\[ P(A \cup B) = P(A) + P(B) - P(A \cap B) \] \[ \]

For minimum \(P(A \cap B)\), assume events are mutually exclusive, i.e. \(P(A \cap B) = 0\)
Check if this is possible:
\[ \] \[ P(A \cup B)_{disjoint} = P(A) + P(B) = \frac{1}{2} + \frac{2}{3} = \frac{7}{6} > 1 \] \[ \]
=> A and B cannot be mutually exclusive

\[ \] \[ P(A \cap B)_{min} = P(A \cup B)_{disjoint} - P(A \cup B)_{max} = \frac{7}{6} - 1 \] \[ \] \[ => P(A \cap B)_{min} = \frac{1}{6}\] \[ \]

Maximum P(A B) is achieved when the events are independent events.

\[ => P(A \cap B)_{max} = P(A)P(B) = \frac{1}{2} * \frac{2}{3} = \frac{1}{3}\]
